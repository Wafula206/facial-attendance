## Features

- ✅ Real-time face recognition using OpenCV
- ✅ Multi-face detection and recognition
- ✅ Automatic attendance marking
- ✅ Check-in/Check-out system
- ✅ Role-based authentication (Admin, Lecturer, Student)
- ✅ Course enrollment management
- ✅ Attendance reports and analytics
- ✅ Responsive design for all devices

## Tech Stack

- **Backend:** Django 4.2
- **Database:** SQLite (local) / PostgreSQL (production)
- **Face Recognition:** OpenCV, face-recognition library
- **Frontend:** HTML5, CSS3, JavaScript

## Deployment

This project is deployed on Render.com

## Local Setup

1. Clone the repository
2. Create virtual environment: python -m venv venv
3. Activate: env\Scripts\activate (Windows) or source venv/bin/activate (Linux/Mac)
4. Install dependencies: pip install -r requirements.txt
5. Run migrations: python backend/manage.py migrate
6. Create superuser: python backend/manage.py createsuperuser
7. Run server: python backend/manage.py runserver
